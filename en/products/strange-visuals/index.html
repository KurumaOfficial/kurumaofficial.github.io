Information

Everything you should know
about the mod

Strange Visuals focuses on visual enhancements that don't violate server rules.

file_download

Installation

Install Minecraft Fabric 1.21.8 in your launcher. Download Fabric API and place the file into the mods folder. After that, download our mod and place it into the mods folder as well. Done, you can launch it.

layers

Compatibility

The mod is compatible with other Fabric mods and works well in modpacks. We recommend using it together with optimization mods and other client-side add-ons.

code

Open Source

The project is distributed with open GPL source code. You can study it, improve it and fork it while keeping the licence and attribution intact.

public

Multiserver support

The mod can be used on servers with different rules. A flexible system for hiding individual features automatically disables whatever is not allowed on a specific server.

workspace_premium
