Принцип

Aleph Studio исходит из того, что пользователь должен понимать, что именно запускается на его компьютере. Поэтому мы максимально придерживаемся open-source подхода и, где это применимо, публикуем код под лицензией GPL: его можно использовать и развивать в собственных проектах при сохранении той же открытой лицензии.

Сохранение и GitHub sync

Параметры GitHub sync

Кнопка «Сохранить для всех» делает commit прямо в файл сайта через GitHub API. Для GitHub Pages укажи путь к опубликованному HTML, обычно index.html. Если нужен только локальный черновик в браузере, используй «Сохранить локально».

Для файла загрузки указывай путь вроде ./files/strange-visuals-latest.jar. Если у продукта есть исходный код, впиши ссылку на репозиторий в поле «Исходный код / GitHub». Формат .json тоже поддерживается, но для сайта в одном файле рекомендуется путь к HTML.
