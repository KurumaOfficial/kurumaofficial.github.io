Принцип

Ты знаешь что
запускаешь

Aleph Studio исходит из того, что пользователь должен понимать что именно работает на его машине. Поэтому мы публикуем код под лицензией GPL — его можно изучить, развить, использовать в своих проектах.

Это не альтруизм. Это стандарт качества.

Как обновить сайт

  1. Открой «Продукты» или «Прочее» и внеси правки.
  2. Нажми «Применить черновик» чтобы увидеть preview.
  3. Для себя — «Сохранить локально».
  4. Для всех — вставь GitHub token и «Сохранить для всех».
