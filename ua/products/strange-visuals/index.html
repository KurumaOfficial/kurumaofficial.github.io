Все, що потрібно знати про мод

Strange Visuals зосереджений на візуальних покращеннях, які не порушують правила серверів.

Встановлення

Встановіть Minecraft Fabric 1.21.8 у своєму лаунчері. Завантажте Fabric API і покладіть файл у папку mods. Після цього завантажте наш мод і також покладіть його у папку mods. Готово, можна запускати.

Сумісність

Мод сумісний з іншими Fabric-модами і добре працює у збірках. Рекомендуємо використовувати його разом із модами на оптимізацію та іншими клієнтськими доповненнями.

Відкритий код

Проєкт поширюється з відкритим GPL-кодом. Код можна вивчати, доопрацьовувати й форкати зі збереженням ліцензії та авторства.

Мультисерверність

Мод може використовуватися на серверах з різними правилами. Гнучка система приховування окремих функцій автоматично вимикає те, що заборонено на конкретному сервері.

Підтримка

Якщо ви хочете допомогти в розвитку мода та підтримати нашу команду, зробити це можна на сторінці підтримки.

Версія Minecraft

Мод розроблений і підтримується для версії 1.21.8 та може перейти на новішу версію в майбутньому.

Fabric API

Для коректної роботи мода потрібні актуальні Fabric Loader і Fabric API під Minecraft 1.21.8.

Проблеми

Якщо у вас виникли проблеми із запуском або ви знайшли баг, можете повідомити про це на нашому Discord сервері.
