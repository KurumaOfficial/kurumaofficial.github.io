Усе, що потрібно знати про мод

Strange Visuals зосереджений на візуальних покращеннях, які не порушують правил серверів.

file_download

Встановлення

Встановіть Minecraft Fabric 1.21.8 у своєму лаунчері. Завантажте Fabric API і покладіть файл у папку mods. Після цього завантажте наш мод і теж перемістіть його в mods.

layers

Сумісність

Мод добре працює з іншими Fabric-модами й підходить для повноцінних збірок. Найкраще він поєднується з модами на оптимізацію та клієнтськими доповненнями інтерфейсу.

code

Open Source

Вихідний код відкритий у публічному репозиторії. Ти можеш переглянути реалізацію, перевірити збірку та використати проєкт як основу для власних експериментів.

public

Готовий до різних серверів

Strange Visuals добре адаптується до різних серверів і режимів. Гнучкі налаштування дозволяють залишити лише ті модулі, які справді потрібні саме тобі.

workspace_premium

Підтримка

Якщо хочеш допомогти розвитку проєкту та нашій команді, це можна зробити тут.

update

Версія Minecraft

Мод розробляється і підтримується для Minecraft 1.21.8. За потреби ми підготуємо оновлення і для новіших версій гри.

build

Корисні модулі

Окрім візуалу, збірка містить практичні дрібниці: гнучкі блоки HUD, допоміжні інструменти та швидке налаштування тих частин інтерфейсу, які для тебе найважливіші.
