Галерея карточек донаторов TEST

Standard tier · $1 – $99

Базовые карточки без специальных эффектов. Цвет — прозрачный с лёгким ruby-оттенком. Бордер: rgba(180, 50, 70, 0.12).

Gold tier · $100 – $499

Ruby-свечение, интенсивность растёт с суммой. Формула glow = (amount − 100) / 400. Чем больше донат, тем ярче аура вокруг карточки.

Platinum tier · $500+

Голубое «платиновое» свечение с металлическим градиентом. Бордер rgba(130, 180, 255, 0.34), радиальный фон.

Top 3 · специальные ранги

Три лучшие карточки помечаются рангом data-rank="1|2|3" — получают усиленные ruby-эффекты: #1 ярче всех, #3 самый мягкий.

Специальные случаи

Длинные имена (должны быть обрезаны многоточием), пустое имя (fallback-аватар без инициала), нулевая сумма, минимальный донат.
