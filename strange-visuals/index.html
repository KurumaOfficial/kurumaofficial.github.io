Что такое
Strange Visuals?

Strange Visuals — форк популярного визуального мода, адаптированный для актуальной версии Minecraft с API Fabric. Мод изменяет визуальную составляющую игры: добавляет новые эффекты частиц и улучшает уже существующие визуальные элементы.

Весь исходный код опубликован под лицензией GPL — его можно изучить, модифицировать и использовать в собственных проектах. Репозиторий доступен на GitHub.
