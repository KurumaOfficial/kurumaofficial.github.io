Как обновить сайт

  1. Открой «Продукты» или «Прочее» и внеси правки.
  2. Нажми «Применить черновик», чтобы увидеть preview.
  3. Для себя — «Сохранить локально».
  4. Для всех — вставь GitHub token и «Сохранить для всех».
